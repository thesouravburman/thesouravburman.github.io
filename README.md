Personal portfolio of **Sourav Burman** — AI & Computer Vision Developer, B.Tech CSE 2023–2027, Brainware University, Kolkata.

Built with pure HTML, CSS, and JavaScript. No frameworks. Hosted on GitHub Pages.

### Features
- Binary rain canvas (hero)
- Magnetic cursor with glow trail
- Letter-by-letter name reveal + typewriter subtitle
- Scroll-triggered animations (GSAP + ScrollTrigger)
- Bento skills grid · Animated timeline · Project cards
